# Pizza Sales  Analysis 

## Project Overview
This project involves analyzing a fictional Pizza Hut dataset to extract meaningful insights regarding sales performance, customer behavior, and product popularity using **MySQL**.

## Key Questions Answered
* **Basic:** Total orders, total revenue, and highest-priced pizzas.
* **Intermediate:** Category-wise distribution, hourly order patterns, and average pizzas per day.
* **Advanced:** Cumulative revenue growth and top 3 pizzas per category by revenue.

## Database Schema
The project utilizes the following tables:
* `pizzas`
* `pizza_types`
* `orders`
* `order_details1`

## How to Run
1. Create a database named `pizzahut`.
2. Import the provided CSV datasets.
3. Execute the `pizza_analysis.sql` script to see the results.
